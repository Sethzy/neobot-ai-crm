# HyperAgent — Competitor Reference

**URL**: https://www.hyperagent.com/
**Logged**: 2026-02-20
**Relevance**: Very close to what Sunder is building

## What It Is
Enterprise agent platform built by the **Airtable team**. Deploy, manage, and scale AI agents powered by frontier models.

## Key Features
- **Ready-to-use agents** with tools: browser, shell, file system, code execution, image/video gen, enterprise integrations
- **Skills** — teach agents reusable workflows (API interactions, frameworks, branded writing) that improve over time via suggested refinements
- **Deployable Roles** — package agent configs (prompts, tools, models, budgets) and deploy to Slack or similar as always-on assistants
- **Fleet Oversight** — real-time monitoring of usage, cost, performance; LLM-as-judge scoring; A/B testing of model/prompt/tool changes

## Positioning
"Intelligence is no longer the bottleneck. Harnessing it is."

Targets the gap between having powerful AI and actually leveraging it systematically — not just chatbots, but compounding, improving agent systems.

## Status
Waitlist / early access only. No public pricing.

## Why This Matters for Sunder
- Skills = our skills/workflows concept
- Roles = our per-client agent configs
- Fleet oversight = our orchestration + monitoring layer
- They're enterprise-up, we're SMB-up via WhatsApp — different GTM, similar product vision
- Built by Airtable alumni — credible team, validates the market
