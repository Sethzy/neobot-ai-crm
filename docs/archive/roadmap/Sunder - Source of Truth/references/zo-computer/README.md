# Zo Computer Reference

Zo Computer (zo.computer) is a personal AI computer platform. Each user gets a persistent Linux server with an AI agent that can run tools, scheduled automations, and publish to zo.space.

**Notable:** Offers MiniMax 2.5 and Kimi 2.5 models for free. Also supports GPT-5.x, Claude Opus 4.6, Gemini 3.1 Pro.

## Components

- `00-source-tool-definitions-verbatim.md` — Complete tool definitions (~61 tools) with full schemas. Categories: file I/O, shell, web browser, web search, media, zo.space, services, agents, personas, rules, app integrations.
- `01-source-runtime-and-architecture-verbatim.md` — Agent runtime model (loop-based execution), persona/rule injection, memory/persistence model, infrastructure model.
